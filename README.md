# 회원가입 페이지
## 핸드폰 인증
- 휴대폰 번호를 입력하고 인증하기를 누르면 밑에 번호를 입력하는 칸이 나옵니다.
- 인증번호는 아무 숫자 6자리를 입력하면 됩니다.
- 시간이 초과하기 전에 인증하기 버튼을 누르면 다시 시작합니다.
- 인증 시간을 초과하면 인증번호 입력창이 사라집니다.

### 추가할만한 기능
- 인증하기 버튼을 누른 후 핸드폰 번호 바꾸기
